# Growfinix Internship Task 1

## Advanced Prompt Engineering for Visuals

### Objective
Generate an AI image using a detailed prompt with the OpenAI Images API.

### Technologies
- Python
- OpenAI API

### How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Add your OpenAI API key in `main.py`.

3. Run:
   python main.py

The generated image will be saved as `generated_image.png`.
